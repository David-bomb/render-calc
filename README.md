# Разработка интернет-приложений (РИП) 2025

## Расчет времени рендера 3D сцены на рендер-ферме

Разработанное приложение позволит художникам создавать и формировать заявки на расчёты времени рендера сцены на различных серверных конфигурациях, а администраторам фермы завершать или отклонять данные заявки.
